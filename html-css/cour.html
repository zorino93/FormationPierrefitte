*{ padding:0; margin:0; }


Sélectionne tous les éléments ayant le nom de class "ma-jolie-nav" :


.ma-jolie-nav{ }


Sélectionne tous les éléments ayant le nom de mon id "mon-unique-id" :

#mon-unique-id{ }

Sélectionne toutes les balises <div>, <article> ET <aside> :


<div> <article> </article> <aside> </aside> </div>


Sélectionne les liens <a> inclus/imbriqués dans la balise <nav> :


<nav> <ul> <li><a href=""></a></li> </ul> </nav>


Sélectionne tous les liens de la nav et active une animation lors du survol de la souris :


a{ } a:hover{ }


Décrivez leur usage et leurs valeurs possibles background:


color: couleur de texte
float: faire flotté un élément d'objet ou de texte
font-family: style de police
font-size: la hauteur de police
height: défini la hauteur
left: gauche
right: droite
line-height: hauteur de la ligne
margin: marge extérieur
padding: marge intérieur
position: Définit le mode de positionnement d'un élément.
text-align: alignement des textes
text-decoration: décoration de texte top: défini la haut
transition: permettent de contrôler la vitesse d'animation.
width: défini la largeur


Propriétés CSS à rechercher Recherchez les propriétés ci-dessous, faites des tests avec sur cette page depuis la console de débuggage pour tester les valeurs qu'elles peuvent prendre :


font-weight: défini la taille du gras de police.
letter-spacing: défini l'espacement des lettres.
word-spacing: défini l'espacement des mots.
text-shadow: ajout d'ombre du texte
background-image: définir une ou plusieurs images en arrière plan.
background-size: taille des images en arrière plan
opacity: défini la transparence d'un élément.
border-style: défini le style des bordure des 4 cotés
box-shadow: défini l'ombre de la boite.
list-style-image: défini l'image d'une puce. border: définir les bordures
border-radius: défini les coins de la bordure
border-width: définir la largeur de la bordure d'une boite.
font-variant: permet de mettre le texte en capitale minuscule.
background-color: défini la couleur utilisé en arrière plan.

Exemple


font-weight: bold;

letter-spacing: 1px;

word-spacing: 4px;

text-shadow: 1px 1px 2px pink;

background-image: linear-gradient(rgba(0, 0, 255, 0.5), rgba(255, 255, 0, 0.5)),

background-size: contain;

opacity: 0.33;

border-style: dotted;

box-shadow: 10px 5px 5px red;

list-style-image: none;

border-radius: 30px;

border-width: thick;

font-variant: normal;

background-color: #74992e;
